# 🛡️ Threat Hunt Manager

A professional MITRE ATT&CK based threat hunting management platform for SOC teams.

## Quick Start
```bash
npm install
npm run dev
```
Open http://localhost:5173

## Features
- Hypothesis tracking and management
- MITRE ATT&CK coverage visualization
- SIEM query management (Splunk SPL, QRadar AQL, Sentinel KQL)
- Excel import and export
- Monthly campaign tracking
- PDF report generation
- Global search
- Mobile responsive

## Tech Stack
- Vite + React
- Tailwind CSS
- Recharts (charts)
- D3.js (coverage map)
- SheetJS (Excel)
- jsPDF (PDF export)
- TanStack Table
- Lucide React (icons)
